# shiftcalc-pro

A sophisticated, high-performance web calculator that seamlessly transitions between **Basic** and **Scientific** modes. Designed with a modern dark-theme aesthetic, this project demonstrates advanced front-end logic, responsive design, and persistent data handling.

---

## 🌟 Key Features

* **Dual-Mode Interface**: Starts as a clean, minimal basic calculator. One click unlocks a full scientific suite including Trigonometry ($sin, cos, tan$), Logarithms ($ln, log$), and Powers ($x^y, \sqrt{x}$).
* **Persistent Calculation Log**: Features an interactive history sidebar that saves your work. Even if you refresh the page or close the browser, your history remains intact thanks to **LocalStorage** integration.
* **Fully Responsive & Mobile-Friendly**: The UI automatically adjusts for desktops, tablets, and smartphones. On mobile, the history panel becomes a full-screen overlay for better accessibility.
* **Smart Display**: Implemented horizontal scrolling for long calculations (e.g., large exponents) to ensure the layout never breaks.
* **Addictive UI/UX**: Built with a "Glassmorphism" inspired dark theme, custom Orbitron typography, and smooth CSS animations.

---

## 🛠️ Tech Stack

* **HTML5**: Semantic structure for accessibility.
* **CSS3**: Advanced Grid and Flexbox for a bulletproof responsive layout.
* **JavaScript (ES6+)**: Custom logic for mathematical evaluations and DOM manipulation.
* **Web Storage API**: Utilized `localStorage` for client-side data persistence.
