# 🚀 ShiftCalc Pro – Smart Scientific Calculator

ShiftCalc Pro is a modern, feature-rich web-based calculator that supports both **basic and scientific calculations** with a clean UI, history tracking, and smart features like voice input and theme switching.

---

## 🌟 Features

### 🔢 Calculator Modes
- Basic mode for simple calculations
- Scientific mode with advanced functions:
  - Trigonometry (sin, cos, tan)
  - Square root, power
  - Constants (π, speed of light)

---

### 🧠 Smart Calculation Logic
- Validates brackets before solving expressions
- Handles floating-point precision issues
- Supports complex mathematical expressions

---

### 📜 Calculation History
- Stores last 20 calculations using localStorage
- Click any history item to reuse it
- Option to clear history

---

### 🌗 Theme Switching
- Dark mode (default)
- Light mode toggle with persistent storage

---

### ⌨️ Keyboard Support
- Supports keyboard input for fast calculations
- Enter → Solve  
- Backspace → Delete  
- Escape → Clear  

---

### 🌡️ Extra Utility
- Temperature conversion (°C → °F)
